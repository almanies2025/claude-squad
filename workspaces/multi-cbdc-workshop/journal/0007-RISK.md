## RISK: Simulation Scenarios Invalidated by CNY Mechanism Error

**Risk ID:** CRITICAL-001
**Category:** Technical/Financial Infrastructure
**Severity:** CRITICAL

### Risk Description

The HKMA dual-issuance claim underpinning scenarios B, E, and F is technically incorrect. HKMA clears CNY via PBOC RTGS; it does not issue CNY. This invalidates:

1. **Scenario B** — CNY dominance via HKMA issuance (wrong mechanism)
2. **Scenario E** — CNY settlement lock (relies on wrong CNY pathway)
3. **Scenario F** — FX Settlement Lock product design (CNY path needs re-architecture)

### Why This Is Critical

The CNY dominance thesis may still hold, but if the simulation's architectural reasoning is wrong, the scenarios produce incorrect results. The product could be designed around a flawed understanding of CNY settlement infrastructure.

### Impact Assessment

| Impact Area          | Severity | Details                                       |
| -------------------- | -------- | --------------------------------------------- |
| Simulation validity  | CRITICAL | Scenarios B, E, F produce wrong results       |
| Product architecture | CRITICAL | FX Settlement Lock CNY path mis-modeled       |
| Business thesis      | MAJOR    | CNY dominance holds but for different reasons |
| Legal/regulatory     | MAJOR    | Wrong representation of HKMA's legal role     |
| Partner confidence   | MEDIUM   | If PBOC/HKMA review,会发现表述错误            |

### Mitigation Required

1. **Immediate:** Revise scenarios B, E, F to use "clearing hub" + "PBOC RTGS access" framing
2. **Short-term:** Re-architect FX Settlement Lock CNY settlement path to model PBOC as sole issuer + HKMA as clearing intermediary
3. **Verification:** Obtain primary source documentation of PBOC-HKMA bilateral agreement

### Residual Risk

If corrected, the CNY dominance thesis likely still holds — Hong Kong's role as offshore clearing hub with direct PBOC RTGS access is still a dominant position. The thesis is justified differently.

**Confidence after mitigation:** MEDIUM-HIGH (requires primary source verification)
